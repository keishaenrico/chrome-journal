your chrome journal 🎨
a little corner of the internet to write about your day — and watch it turn into color.
type out how you're feeling. hit paint. see your emotions bloom into a watercolor painting, unique to you and only you.
built with a single html file, a serverless proxy, and the anthropic api doing the emotional heavy lifting.
